## Steps to creation
1. Create App 
  * npx create-react-app react-qubit-tv
2. Add Sass 
  * yarn add sass-loader node-sass --dev
3. Run Eject ( for adding babel, loaders, eslint, jest, and others)
  * commit changes 
  * npm run eject
4. Add bootstrap


## Steps for run

1. yarn install 
2. yarn start
